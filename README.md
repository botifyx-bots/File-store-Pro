## рҹ“Ӣ бҙҸбҙ бҙҮКҖбҙ ЙӘбҙҮбҙЎ

A **highly secure, multi-featured Telegram File Share Bot** that generates permanent sharing links for files stored in database channels. Built with Pyrogram v2 and MongoDB вҖ” everything is configurable directly from the bot with no restarts needed.

---

### вҖәвҖә КҖбҙҮЗ«бҙңбҙҮsбҙӣ Т“бҙҸКҖбҙ„бҙҮ sбҙңКҷ
By enabling Request Force-Sub mode, users are provided with a private channel link along with a join request. This feature adds versatility, allowing for greater flexibility in managing Force-Sub channels based on individual preferences.

### вҖәвҖә бҙ„бҙңsбҙӣбҙҸбҙҚ Т“бҙҸКҖбҙ„бҙҮ sбҙңКҷ
Add unlimited Force-Sub channels or clear them all instantly вҖ” no limit on how many you can add. Full control with no restrictions.

### вҖәвҖә бҙҖбҙ…бҙ… бҙҚбҙңКҹбҙӣЙӘ бҙҖбҙ…бҙҚЙӘЙҙs
Add or remove multiple admins via user ID. Admins get access to a curated set of bot commands without full owner privileges.

### вҖәвҖә КҷбҙҖЙҙ / бҙңЙҙКҷбҙҖЙҙ
Block spammers and abusive users from accessing the bot instantly. Unban them anytime from the admin panel.

### вҖәвҖә бҙҖбҙңбҙӣбҙҸ бҙ…бҙҮКҹбҙҮбҙӣбҙҮ
Protect your bot from copyright strikes with a smart auto-delete system. Set custom timers вҖ” files are removed automatically, and a retrieval link is sent so users never lose access.

### вҖәвҖә бҙ„бҙҸЙҙбҙӣбҙҮЙҙбҙӣ КҷбҙңбҙӣбҙӣбҙҸЙҙs
Attach fully customizable inline buttons to every shared file. Set button labels and links (e.g. "Join Channel") tailored to your needs.

### вҖәвҖә КңЙӘбҙ…бҙҮ бҙ„бҙҖбҙҳбҙӣЙӘбҙҸЙҙ & бҙҳКҖбҙҸбҙӣбҙҮбҙ„бҙӣ бҙ„бҙҸЙҙбҙӣбҙҮЙҙбҙӣ
Toggle captions on/off for shared files. Enable Protect Content to prevent users from forwarding or screenshotting files.

### вҖәвҖә sбҙӣбҙҖКҖбҙӣ & Т“sбҙңКҷ бҙҚбҙҮбҙ…ЙӘбҙҖ
Provide image URLs or directly send photos to customize the Start and Force-Subscribe messages.

### вҖәвҖә Т“бҙңКҹКҹКҸ бҙҮбҙ…ЙӘбҙӣбҙҖКҷКҹбҙҮ бҙҚбҙҮssбҙҖЙўбҙҮs
Customize Start, About, Reply, and FSub messages with full HTML formatting and dynamic placeholders.

### вҖәвҖә бҙ„бҙңsбҙӣбҙҸбҙҚ бҙңКҖКҹ sКңбҙҸКҖбҙӣбҙҮЙҙбҙҮКҖ
Integrate your own shortener URL and API key with built-in token verification. Includes tutorial links for easy setup.

## вҡҷпёҸ бҙ бҙҖКҖЙӘбҙҖКҷКҹбҙҮs

Edit `config.py` directly before deploying:

```python
# в”Җв”Җ Bot Core в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
SESSION  = "your_session_name"
TOKEN    = "your_bot_token"
API_ID   = your_api_id
API_HASH = "your_api_hash"
WORKERS  = 5

# в”Җв”Җ Database в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
DB_URI   = "your_mongodb_uri"
DB_NAME  = "your_database_name"

# в”Җв”Җ Channels в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
# Force Subscription: [channel_id, request_enabled, timer_in_minutes]
FSUBS      = [[-1001234567890, True, 10]]
DB_CHANNEL = -1001234567890

# в”Җв”Җ Settings в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
AUTO_DEL    = 300          # Auto delete timer in seconds (0 = off)
ADMINS      = [123456789]  # Admin user IDs
DISABLE_BTN = True         # Hide content buttons
PROTECT     = True         # Block forwarding & screenshots

# в”Җв”Җ URL Shortener в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
SHORT_URL = "shrinkme.io"
SHORT_API = "your_api_key"
SHORT_TUT = "https://t.me/your_tutorial_link"
```
